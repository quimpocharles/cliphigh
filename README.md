# SiKAT Zamboanga Highlight Pipeline

Automated basketball highlight generator for Zamboanga SiKAT (MPBL) and other
FIBA-based leagues. Reads live play-by-play data from FIBA LiveStats, cuts a
clip for every scoring play from a YouTube stream or VOD, and compiles
per-quarter highlight reels.

---

## How It Works

```
FIBA LiveStats feed
      ↓
  Detect scoring play (tno, game clock)
      ↓
  Map game clock → video timestamp (piecewise linear interpolation)
      ↓
  Cut clip with ffmpeg (lead + tail seconds around the basket)
      ↓
  Verify clip with Whisper audio transcription
      ↓
  Compile per-quarter highlight reel
      ↓
  Save to highlights/{league}/{team}/{opponent}/{quarter}/
```

- [openai-whisper](https://github.com/openai/whisper)

```bash
pip3 install -r requirements.txt
pip3 install "numpy<2"   # required for Whisper/PyTorch compatibility
```

## Quick Start

### New game setup

```bash
python3 new_game.py
```

Prompts for league, game ID, team, opponent, YouTube URL, and tip-off time.
Auto-detects team TNO from the FIBA feed. Writes a fresh `config.py`.

### VOD / completed game (recommended workflow per quarter)

```bash
# Step 1 — see estimated timestamps
python3 vod_replay.py --dry-run --quarters 1

# Step 2 — verify interactively, anchors saved to config.py automatically
python3 vod_replay.py --calibrate --quarters 1

# Step 3 — cut clips and compile highlight reel
python3 vod_replay.py --quarters 1 --skip-download
```

Each step prints the next command to run.

### Live game

```bash
python3 pipeline.py --game-id 2836518
```

Polls the FIBA feed every 20 seconds, records the YouTube stream, cuts clips
in real time, and compiles the highlight reel at the end of each quarter.

### Other options

```bash
# Process specific quarters only
python3 vod_replay.py --quarters 2 3 --skip-download

# Stop before the stream cuts out
python3 vod_replay.py --quarters 4 --min-clock 03:05 --skip-download
```

## Calibration

Timestamp accuracy depends on **calibration anchors** — confirmed
`(video_seconds, quarter, game_clock)` triples added to `config.py`.

The `--calibrate` mode handles this interactively:
- Shows each play's estimated video timestamp
- Asks you to confirm or correct it
- Validates that timestamps are chronologically consistent
- Saves confirmed anchors to `config.py` automatically

More anchors = more accuracy. Free throws make ideal dead-ball anchors.

## Configuration

All settings are in `config.py` (generated by `new_game.py`):

| Setting | Description |
|---|---|
| `GAME_ID` | FIBA LiveStats numeric game ID |
| `LEAGUE` | mpbl / uaap / ncaa / jr_mpbl |
| `TEAM` / `TEAM_TNO` | Your team's folder name and feed number |
| `OPPONENT` | Opponent folder name |
| `YOUTUBE_STREAM_URL` | YouTube VOD or live stream URL |
| `TIPOFF_VIDEO_SECONDS` | Seconds into the video where tip-off occurs |
| `CALIBRATION_ANCHORS` | Confirmed timestamp anchors |
| `CLIP_LEAD_SECONDS` | Footage before each basket (default 3.5s) |
| `CLIP_TAIL_SECONDS` | Footage after each basket (default 8s) |
| `AUDIO_VERIFY` | Enable/disable Whisper verification |
